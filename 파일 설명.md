# 일로온 AI 챗봇 코드 인수인계 문서

## 개요

일로온 AI 챗봇 기능 구현 코드입니다.

사용자 질문을 입력받아 질문 의도를 분류한 뒤 관련 데이터를 검색하고 Claude를 이용하여 최종 응답을 생성합니다.

현재 구현된 대표 기능은 다음과 같습니다.

* 사용자 질문 응답(사용자 맞춤 응답 가능)
* 부산 지역 공고 데이터 기반 채용 공고 검색
* 사용자 설문 기반 맞춤 공고 추천
* IT 직무 트렌드 안내
* 기업 정보 안내
* 청년 정책 질문 응답(온통청년 Open API + 부산 공공데이터 CSV 2개 활용)
* 퀵리플라이 기능(버튼 클릭시)
* History 기반 공고 후속 질문 처리

## rag.py

챗봇 핵심 로직이 구현된 파일입니다.

주요 기능

### Intent 분류

질문을 아래 Intent 중 하나로 분류합니다.

```text
CUSTOM
GENERAL
TREND
COMPANY
PUBLIC_DATA
CAREER_TIP
INVALID
```

### 개체명 추출

질문에서

```text
직무
지역
회사명
기술스택
```

등의 정보를 추출합니다.

### 공고 검색

사용자 질문과 설문 정보를 기반으로 관련 공고를 검색합니다.
부산 공고 데이터의 position.job_category.mid 값을 활용하여 직무 카테고리를 매핑합니다.
회사명/공고명 기반 상세 검색을 지원하며, 사용자가 이전 추천 결과를 바탕으로 “세 번째 공고 자세히 알려줘”, “그 공고 연봉 알려줘”와 같이 질문할 경우 history를 기반으로 검색 질의를 재작성하여 해당 공고를 다시 조회합니다.

### 트렌드 검색

직무별 트렌드 데이터를 검색합니다.

### 정책 추천

온통청년 Open API와 부산광역시 청년지원정책 현황 CSV, 청년일자리지원 서비스 CSV를 함께 활용합니다.

현재 구현 내용

* 취업 정책 추천
* 창업 정책 추천
* 지원 정책 및 기관 추천
* 정책명 직접 검색

예시

```text
국민취업지원제도 알려줘
```

입력 시 정책명을 직접 검색하여 상세 내용을 안내합니다.

정책 추천 시에는 API 결과를 기반으로 관련 정책을 선별하여 응답합니다.

### Claude 응답 생성

검색된 결과를 기반으로 Claude API를 사용하여 최종 답변을 생성합니다.

# 연동 시 확인 사항

## user_info

백엔드에서 사용자 설문 데이터를 조회한 뒤 user_info 객체로 전달하는 방식으로 구현되어 있습니다.

예시

```json
{
  "message": "나에게 맞는 공고 추천해줘",
  "user_info": {
    "job_type": "개발",
    "region": "부산"
  }
}
```

# 참고 사항

* 개발 및 테스트는 FastAPI 환경에서 진행하였습니다.
* Claude API Key와 Voyage API Key가 필요합니다.
* 정책 추천 기능은 rag.py 내부에 구현되어 있습니다.
* 정책 추천 기능은 온통청년 Open API와 부산 공공데이터 CSV를 함께 사용합니다.
* 공고 데이터는 기존 직무별 더미 공고에서 부산 지역 공고 데이터로 변경되었습니다.
* history를 활용하여 이전 추천 결과에 대한 후속 질문을 처리할 수 있습니다.
